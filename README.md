# Dev Proxy (dev-proxy)

Dev Proxy is a Microsoft command-line tool for simulating, testing, and debugging API interactions during development. It is an API simulator that helps developers test their apps beyond the happy path: simulating errors, rate limits, and slow responses, mocking APIs without writing code, and providing contextual guidance for Microsoft Graph and other APIs. Dev Proxy is open source, cross-platform, and free to use.
